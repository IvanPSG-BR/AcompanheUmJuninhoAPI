# Requisitos

## Funcionais

- CRUD de Usuário
- Autenticação de Usuário
- Escolha entre ser Juninho ou Mentor
- (Juninho) Match para encontrar os mentores mais adequados
- (Mentor) Visualização dos Juninhos decididos pelo matchmaking para escolha
- Espaço de troca de mensagens
- Diário de bordo para registrar o progresso e próximos passos

## Não Funcionais

- Respostas rápidas
- Segurança contra ataques hackers
- Segurança dos dados do usuário (privacidade)
